# Example Inputs And Outputs

## Example 1

### User Input

> My grandma sent this link and says everyone over 60 must stop eating tofu. Can you use elder-factcheck?

### Expected Behavior

- classify as `Health And Wellness`
- identify the main nutritional or medical claims
- check whether the claim is supported by reliable medical guidance
- avoid shaming the grandmother
- produce a gentle reply suggestion

## Example 2

### User Input

> My dad forwarded this screenshot saying a new pension rule starts next week. Is it real?

### Expected Behavior

- classify as `Finance And Consumer Safety` or `Politics And Public Policy`
- verify with official policy or banking sources if available
- distinguish between confirmed information and reposted rumor
- suggest a direct but respectful reply for the father

## Example 3

### User Input

> My mom sent this article saying children who don't visit their parents are all unfilial. Help me reply without sounding confrontational.

### Expected Behavior

- classify as `Emotion, Relationships, And Aging`
- focus more on framing and manipulation than literal factual verification
- identify absolutist and guilt-based rhetoric
- produce an affectionate but grounded reply

## Example 4

### User Input

> My grandpa sent a screenshot warning that all bank cards will be frozen if we don't register something today.

### Expected Behavior

- classify as `Finance And Consumer Safety`
- treat urgency as a warning sign
- verify with official banking or regulator sources when possible
- recommend caution and verification before action
